# Bot Telegram Samware
Por ahora está muy verde y desprolijo, pero es para tener algo andando. Se aceptan contribuciones.
Para correrlo se necesita el archivo `.env` en el root, y definir `BOT_TOKEN` como indica la [documentación](https://core.telegram.org/bots/tutorial#getting-ready) de Telegram. La idea del bot es centralizar el acceso a la información e ir agregando herramientas y cosas útiles.
